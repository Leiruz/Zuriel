About Me

Cybersecurity

I first started with developing my first 3 FA system (MD5, SHA256 & OTP) using python when I was 15. Ever since, I worked on many cyber-related project such as creating ASOC system with open-source technology, subject matter expert on Carbon Black and CyberArk, did web application pentesting, android & bluetooth pentesting, did RE on malwares, work on Secure Software Dev as well as intern at NCS for 6 months.

Artificial Intelligence

Took up Deep Learning module at Ngee Ann Polytechnic where I wrote 2 papers on Convulutional Neural Network on food image classification and Recurrent Neural Network on autocorrect and emoji prediction. Currently, I am working on creating a deep learning model for malware prediction
